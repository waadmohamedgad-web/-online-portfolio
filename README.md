# Portfolio Starter (GitHub Pages)

This is a simple starter template for a personal portfolio website.

## What you need to edit
1. **Your name** in `index.html`
2. Profile links (GitHub / LinkedIn / Email)
3. Project cards (title, description, links, screenshots)
4. Replace the placeholder image:
   - `assets/profile-placeholder.svg` → your photo (e.g., `profile.jpg`)
5. Replace `assets/cv.pdf` with your real CV

## Run locally
Just open `index.html` in your browser.

## Publish on GitHub Pages
1. Create a new repo on GitHub (example: `username.github.io` or `portfolio`)
2. Upload the files or push using Git
3. Go to **Settings → Pages**
4. Select **Deploy from a branch**
5. Choose branch: `main` and folder: `/root`
6. Save — your site will be live in a minute

## Notes
- The template includes **Dark/Light mode** and a simple **EN/AR toggle**.
- The contact form uses `mailto:` (no backend required).
